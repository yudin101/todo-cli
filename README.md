# todo-cli

Todo CLI - A simple task manager

**Clone the project**
```bash
git clone https://github.com/yudin101/todo-cli.git
```

**Go to project directory**
```bash
cd todo-cli
```

## Installation
To build and install `todo` in `~/.local/bin`

```bash
make
make install
```

## Usage
```bash
todo help
```

## Removal

|Command|Action|
|-------|------|
|make clean|Remove object files and compiled binary from source directory|
|make uninstall|Remove binary from `~/.local/bin`|
|make purge|`clean`, `uninstall`, also remove data|
